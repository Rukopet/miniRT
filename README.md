## Installation

#### What we need
 - On Mac OS : clang/gcc

 Unfortunately I have not tested my project on other systems, sorry, **only mac** :pensive:
 
#### Compile & clone
```shell
git clone https://github.com/Rukopet/miniRT
```
```shell
make
```
## Usage
```shell
./miniRT [filescene.rt]
```
